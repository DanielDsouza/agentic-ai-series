## 🚀 Getting Started

1. Clone this repo:
   ```bash
   git clone https://github.com/<your-username>/agentic-ai-series.git
   cd agentic-ai-series
   ```

2. Navigate to the article folder you want to try:
   ```bash
   cd 01-what-is-agentic-ai
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Add your API key and run:
   ```bash
   python agent_calculator.py
   ```
